func Create(name string) (*File, error) 创建文件

如果文件存在则会清除文件内容，文件权限为0666（umask之前：所谓的umask就是在当前文件夹的现有权限上减去umask指定的权限，即为新创建的文件权限），文件File拥有O_RDWR读写权限。

f, err := os.Create("a.txt")

// ls -l a.txt
-rw-r--r--  1 xxx  wheel  0  6 25 14:24 a.txt
func (f *File) Truncate(size int64) error* 截取文件

func Truncate(name string, size int64) error

改变文件大小，但是不改变IO起点，如果文件不足指定的大小，则以null填充，如果超过，则将文件大小截取到指定大小。

os.Truncate("a.txt", 10)

// 不足10B时的内容
-rw-r--r--  1 fujiangkun  wheel    10B  6 25 14:30 a.txt

// 超过10B时的内容，内容会截断
-rw-r--r--  1 fujiangkun  wheel    14B  6 25 14:31 a.txt

func Open(name string) (*File, error) 文件打开

func OpenFile(name string, flag int, perm FileMode) (*File, error)

open打开的文件只有O_RDONLY只读权限，openFile可以设置读写权限和文件权限。文件权限包含以下几种，多种格式可以使用|或进行拼接。如os.O_APPEND | os.O_CREATE

权限值 释义
os.O_CREATE 如果文件不存在则先创建
os.O_APPEND 往文件中添建（Append）
os.O_RDONLY 只读
os.O_WRONLY 只写
os.O_RDWR 读写
os.O_TRUNC 文件打开时裁剪文件
os.O_EXCL 和O_CREATE一起使用，文件不能存在
os.O_SYNC 以同步I/O的方式打开

func ReadDir(dirname string) ([]os.FileInfo, error) 文件夹

返回dirname文件夹下所有文件信息，文件信息列表按照文件名升序排序。

fs, err := ioutil.ReadDir("/")

  panic(err)

for _, f := range fs {

func ReadFile(filename string) ([]byte, error) 文件读取

读取文件全部内容，正因为读取的是文件全部内容，读取成功返回的err是nil，而不是io.EOF错误。

bs, err := ioutil.ReadFile("a.txt")
if err != nil {
  panic(err)
}
fmt.Println(string(bs))

func TempDir(dir, prefix string) (name string, err error) 创建临时目录

当dir路径为""时，使用系统的临时目录，如mac的/tmp，使用prefix作为目录名称的前缀，后面加上一个随机数字串。调用方需要在不需要临时文件夹的时候将其清除。

// 在a目录下创建一个前缀为tmp的文件夹
pth, err := ioutil.TempDir("a", "tmp")
if err != nil {
  log.Fatal(err)
}
defer os.RemoveAll(pth)
fmt.Println(pth)
func TempFile(dir, pattern string) (f *os.File, err error) 创建临时文件

在文件夹内创建一个临时文件，获取一个拥有读写权限的File对象，文件使用随机数串拼接在pattern字符串后面，如果字符串pattern中含有*，则将最后一个*用随机数字串替换，如果dir为空，则使用默认的系统临时文件夹\tmp，需要调用方维护文件，在不需要的时候将其删除。

content := []byte("temporary file's content")
// tmpxxxxxxxxx
tmpfile, err := ioutil.TempFile("a", "tmp")
if err != nil {
	log.Fatal(err)
}
defer os.Remove(tmpfile.Name()) // clean up

// tmp-xxxxxxxxx.txt
tmpfile, err := ioutil.TempFile("a", "tmp-*.txt")
