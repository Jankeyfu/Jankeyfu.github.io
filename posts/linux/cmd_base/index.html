文件目录命令

ls (list directory content) 列出当前目录下的文件和文件夹
ls		#只展示文件名
ls -l	#展示文件名即其他文件信息，包括权限，日期等
ls -a	#展示隐藏文件
cd (change directory)跳转到指定目录路径
cd dir_name # 跳转到指定的目录路径下
~ 			# 跳转到用户目录下
.. 			# 跳转到上一级目录
. 			# 当前目录
